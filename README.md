# VIP Movie Analysis Tool

## Description
This project is a user script designed to enhance the viewing experience on various video platforms by providing quick access to movie analysis and search functionalities. It supports multiple languages and integrates with popular video and search platforms.

## Table of Contents
- [Installation](#installation)
- [Usage](#usage)
- [Features](#features)
- [Supported Platforms](#supported-platforms)
- [Contributing](#contributing)
- [License](#license)

## Installation
To install the user script, follow these steps:
1. Install a user script manager like Tampermonkey or Greasemonkey.
2. Add the script to your user script manager.

## Usage
1. Navigate to a supported video platform.
2. The script will automatically enhance the page with additional functionalities.
3. Use the provided buttons and menus to access movie analysis and search features.

## Features
- **Multi-language Support**: Available in multiple languages including English, Chinese, and Traditional Chinese.
- **Quick Search**: Provides quick search options for Douban, Time, and other movie databases.
- **Video Analysis**: Offers detailed analysis of movies and short videos.
- **Search Engine Integration**: Integrates with popular search engines like Google, Bing, and Baidu.

## Supported Platforms
- **Video Platforms**: Youku, QQ Video, iQiyi, LeTV, PPTV, 1905, Mango TV, Bilibili, Fun TV, Sohu, YouTube, Xigua, Douyin, Kuaishou, Xiaohongshu, Weibo, AcFun.
- **Search Engines**: Baidu, Bing, Quark, Sogou, Google.
- **Movie Databases**: Douban, Time.
